# lunarDocs

H4ck3r public mkdocs with [Material for MkDocs](https://squidfunk.github.io/mkdocs-material/), work in progress...

Just learning about machines

## Clone

```bash
git clone git@github.com:rnek0/lunarDocs.git
```

## Write & serve

```bash
mkdoc serve
```

## Build

```bash
mkdoc build
```

## Push

```bash
git pull
```
